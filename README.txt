# Hobby Web App project

This project uses a database, made through Spring, which is then used as the base for a web app using that data, being aple to manipulate it through API calls.

## Getting Started

1: Clone or fork this repository
2: Intitalise it in your own github repsoitory
3: Open the project in a compatible IDE
4: Run the main app class to run the Spring app
5: Open the web app in your localhost at the port specified in the app.properties file

### Prerequisites

```
You will need to install Maven on your device for this project.
you will also need MySQL 5.7 if you want to connect this project locally, by defualt it uses Google cloud platform.

## Running the tests

make sure the app in not running when running the main app class test, all the other Unit and integration tests should work fine with the app running
